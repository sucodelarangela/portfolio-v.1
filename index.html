02. Projetos em destaque

AdoPet

A AdoPet é uma página de uma empresa fictícia que funciona como intermediária entre adotantes de pets e ONGs com foco em retirar animais das ruas. A proposta da AdoPet é trazer mais usuários e aumentar a interação entre possíveis adotantes e representantes dessas ONGs e para isso, o objetivo do projeto é implementar uma plataforma e levar a empresa pro mundo digital.
Este projeto foi desenvolvido como resultado de um Desafio Front-End (4ª edição) exclusivo para alunos da escola de tecnologia Alura.

#React #CSS #JavaScript
#ReactHookForm #FramerMotion #HeadlessUI

Alura Geek (E-commerce)

Página de e-commerce (versão do administrador) para uma loja online de produtos geek chamada Alura Geek, desenvolvida como resultado de um Desafio Front-End (3ª edição) exclusivo para alunos da escola de tecnologia Alura.
Escolhi desenvolver este projeto utilizando Node.js com Express e o banco de dados relacional SQLite3, além das linguagens próprias deo front-end, como HTML, CSS, JavaScript e a engine de visualização EJS para dinamicidade na página. Também utilizei a biblioteca Simple Mask Money para validação e estilização do campo de preços dos produtos.

#HTML #EJS #CSS #JavaScript
#Node.Js #Express #SQLite3 #SimpleMaskMoney #Ecommerce

Orange Share

Aplicação web desenvolvida durante um hackathon de 15 dias promovido pelo Grupo FCamara na terceira edição do seu Programa de Formação. O Orange Share tem como proposta ser uma plataforma de busca simples e direta, que possibilita ao colaborador encontrar, de forma rápida e fácil, colegas mais experientes em determinada tecnologia que possam auxiliá-lo com dúvidas rápidas ou, até mesmo, uma mentoria de carreira. O projeto, que se trata de um MVP (Minimum Viable Product), foi desenvolvido em equipe, composta de três Desenvolvedoras e três UX Designers, sendo implementado no prazo estipulado, utilizando Node.js, Express, JavaScript, EJS, CSS, Sequelize e o banco de dados relacional PostgreSQL.
Ao final do programa, cada participante recebeu um certificado de participação. Você pode baixar meu certificado clicando aqui.

#EJS #CSS #JavaScript #Node.Js
#Express #Sequelize #PostgreSQL
